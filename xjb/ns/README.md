# xjc: catalog + bindings

Файлы для **xjc**: `catalog.xml` (OASIS) и `bindings.xjb` (JAXB). Имя подкаталога совпадает с **`namespace-forest/schemas/<версия>/`** (submodule в корне репозитория).

Меняете версии — правьте **`gradle.properties`**, **`SchemaVersion.java`** и при необходимости добавляйте каталог здесь.
